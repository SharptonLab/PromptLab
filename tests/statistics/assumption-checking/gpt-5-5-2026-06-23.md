## Assumption 7: Bray–Curtis distance-space issues

### Plain-language meaning

Bray–Curtis is not fully Euclidean. This can produce negative eigenvalues in PCoA. Usually this does not invalidate PERMANOVA, but it can affect ordination and dispersion calculations.

### How to check

```r
pcoa_check <- wcmdscale(bc, eig = TRUE, add = FALSE)

eig <- pcoa_check$eig
sum_negative <- sum(abs(eig[eig < 0]))
sum_positive <- sum(eig[eig > 0])

negative_ratio <- sum_negative / sum_positive
negative_ratio
```

### Interpretation

- Very small negative eigenvalues: not a major concern.
- Larger negative eigenvalues: consider square-rooting distances or using a correction.

### What to do if problematic

Use square-root distances in `adonis2()`:

```r
adonis2(
  bc ~ Treatment * Time,
  data = meta,
  permutations = 9999,
  by = "terms",
  sqrt.dist = TRUE
)
```

Or use an additive correction:

```r
adonis2(
  bc ~ Treatment * Time,
  data = meta,
  permutations = 9999,
  by = "terms",
  add = "lingoes"
)
```
