## 2. Assumption-by-Assumption Guide

### Assumption 1: Correct Permutation Scheme (Exchangeability)

**Plain language**: Under the null hypothesis, sample labels must be freely exchangeable. With repeated measures or nested designs, free permutation is invalid — you'd shuffle dependencies away.

**Key question for you**: Are the same subjects measured at both time points? If yes, you have a repeated-measures structure and must constrain permutations within subject.

**How to handle**: Use `how()` from `permute` package to define permutation strata.

**If violated (wrong scheme)**: p-values are meaningless. Always specify the design explicitly rather than relying on defaults.

---

### Assumption 2: Homogeneity of Multivariate Dispersions

**Plain language**: Each group should have similar within-group variability (spread in multivariate space). PERMANOVA tests centroids; if spreads differ, a "significant" result might reflect dispersion differences rather than location differences.

**How to test**:
- **PERMDISP** (`betadisper` + `permutest`): formal test of equal dispersions
- **Visual**: boxplots of distance-to-centroid by group; PCoA with convex hulls/ellipses

**How to interpret**:
- Significant PERMDISP + significant PERMANOVA → ambiguous; could be location, dispersion, or both
- Non-significant PERMDISP + significant PERMANOVA → cleaner interpretation as location shift
- With **balanced designs**, PERMANOVA is fairly robust to dispersion heterogeneity; with unbalanced, it's badly biased (larger group with smaller dispersion → inflated Type I)

**If violated**:
- Report PERMDISP results alongside PERMANOVA
- Consider distance-based tests less sensitive to dispersion (e.g., **ANOSIM** is *more* sensitive — don't switch to it)
- Use **`anosim`** alternatives carefully; better: report both centroid and dispersion effects as biologically meaningful
- Consider transformations of the community matrix (e.g., Hellinger, log) before computing Bray-Curtis
- For very unequal dispersion, **distance-based redundancy analysis (dbRDA)** with explicit dispersion modeling may help

---

### Assumption 3: Independence of Observations

**Plain language**: Each sample is independent given the model. Repeated samples from the same subject, cage, tank, or batch are non-independent.

**How to check**: This is a *design* assumption — check your sampling protocol, not the data.

**If violated**: Use `strata =` argument in `adonis2` to restrict permutations within blocks (subjects, cages). For your time design, this is critical if subjects were sampled at both times.

---

### Assumption 4: Appropriate Distance Metric

**Plain language**: Bray-Curtis is good for compositional abundance data but is sensitive to rare taxa and total abundance differences.

**How to check**:
- Look at total counts per sample — wildly variable totals can distort Bray-Curtis
- Compare results using alternative metrics (Jaccard for presence/absence, Aitchison for compositional, weighted UniFrac if phylogeny available)

**If problematic**: Rarefy or normalize (e.g., CSS, TSS, or use Aitchison distance via CLR transform). Confirm robustness by re-running with a second metric.

set.seed(123)
permutest(disp_trt,  permutations = 999, pairwise = TRUE)
permutest(disp_time, permutations = 999)
