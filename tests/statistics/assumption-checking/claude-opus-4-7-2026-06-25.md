## 2. Each Assumption in Detail

### Assumption 1: Independence / Exchangeability

**Plain language:** Each sample is independent of the others, OR you've correctly specified the dependency structure (blocks, strata, repeated measures).

**How to check:**
- Not a statistical test—comes from study design
- Ask: Were samples collected from the same subject over time? Spatial autocorrelation? Tank/cage effects?

**Your case:** You have Time as a factor. If the *same* subjects were sampled at both timepoints, you have repeated measures and must restrict permutations.

**Fix if violated:**
```r
# Restrict permutations within subject for repeated measures
library(vegan)
perm <- how(nperm = 999)
setBlocks(perm) <- with(metadata, subject_id)
adonis2(bray_dist ~ Treatment * Time, data = metadata, permutations = perm)
```

---

### Assumption 2: Homogeneity of Multivariate Dispersions

**Plain language:** Each group has similar "spread" or variability around its centroid in multivariate space. PERMANOVA tests whether group *centroids* differ, but with unequal dispersions a significant result could reflect spread differences instead.

**How to test:**
```r
library(vegan)

# Test dispersion homogeneity for each factor
disp_trt <- betadisper(bray_dist, metadata$Treatment)
disp_time <- betadisper(bray_dist, metadata$Time)
disp_group <- betadisper(bray_dist, interaction(metadata$Treatment, metadata$Time))

# Formal test
permutest(disp_trt, permutations = 999)
permutest(disp_group, permutations = 999)

# Pairwise comparisons
TukeyHSD(disp_trt)

# Visual diagnostics
plot(disp_trt)                    # PCoA with hulls/ellipses
boxplot(disp_trt)                 # Distance-to-centroid boxplots
```

**Interpretation:**
- `permutest` p < 0.05 → significant dispersion heterogeneity
- Examine the boxplot: how big is the difference in mean distance-to-centroid?

**What to do if violated:**

| Situation | Recommendation |
|-----------|----------------|
| Balanced design (your case: n=10 each) + moderate heterogeneity | **Proceed with PERMANOVA**—Anderson & Walsh (2013) showed it's robust |
| Unbalanced + heterogeneous | Report dispersion test alongside; interpret cautiously |
| Strong heterogeneity, want to disentangle | Use **both** PERMANOVA and PERMDISP; report both |
| Want location-only inference | Consider **NPMANOVA on rank-transformed distances** or **ANOSIM** (though ANOSIM is more sensitive to dispersion, not less) |

## 3. Complete R Workflow

```r
library(vegan)

# ---- 0. Setup ----
bray <- vegdist(community, method = "bray")
table(metadata$Treatment, metadata$Time)   # Check balance

# ---- 1. Dispersion checks ----
disp_trt   <- betadisper(bray, metadata$Treatment)
disp_time  <- betadisper(bray, metadata$Time)
disp_cell  <- betadisper(bray, interaction(metadata$Treatment, metadata$Time))

permutest(disp_trt,  permutations = 999)
permutest(disp_time, permutations = 999)
permutest(disp_cell, permutations = 999)

par(mfrow = c(1, 2))
plot(disp_trt, main = "Treatment dispersion")
boxplot(disp_trt, main = "Distance to centroid")

# ---- 2. PERMANOVA with proper permutation scheme ----
# If independent samples:
set.seed(123)
res <- adonis2(bray ~ Treatment * Time,
               data = metadata,
               permutations = 999,
               by = "terms")   # or by = "margin" for Type III-like
res

# If repeated measures (same subject across Time):
perm <- how(nperm = 999)
setBlocks(perm) <- with(metadata, subject_id)
res_rm <- adonis2(bray ~ Treatment * Time, data = metadata, permutations = perm)

# ---- 3. Sensitivity analysis with alternative distance ----
hell <- vegdist(decostand(community, "hellinger"), method = "euclidean")
adonis2(hell ~ Treatment * Time, data = metadata, permutations = 999)
```
