````
### Assumption 1: Independence of Observations
#### What it means:
Samples must be independent for permutation tests to generate a valid null distribution. Your design has a `Time` predictor: if `Time` is a *within-subject* (repeated) measure (same individuals sampled at both time points), samples from the same subject are correlated and violate independence. Non-independence also arises from spatial/temporal autocorrelation (samples close in space/time are more similar) or unaccounted batch effects (samples processed in the same batch are more similar).
#### How to test:
This is primarily a design-based check, with supporting diagnostics:
1. **Design check**: Confirm if `Time` is repeated (same subjects across time points) or independent (different subjects at each time point). If you have a `Subject` ID column with 30 unique IDs (10 per Treatment × 2 time points), you have repeated measures.
2. **Repeated measures diagnostic**: Compare within-subject distances (same subject, different time points) to between-subject distances (different subjects, same time point):
```r
# Only run if you have a Subject ID column
if("Subject" %in% colnames(meta)) {
  # Calculate within-subject Bray-Curtis distances
  within_dist <- unlist(lapply(unique(meta$Subject), function(sub) {
    sub_samples <- meta[meta$Subject == sub, ]
    if(nrow(sub_samples) == 2) {
      idx <- match(sub_samples$SampleID, rownames(bc_dist))
      as.numeric(bc_dist[idx[1], idx[2]])
    }
  }))
  # Calculate between-subject distances at Time 1 (example)
  time1_idx <- which(meta$Time == "Time1")
  between_dist <- as.dist(bc_dist[time1_idx, time1_idx])
  # Visualize
  dist_df <- data.frame(
    Distance = c(within_dist, between_dist),
    Type = rep(c("Within-subject", "Between-subject"), c(length(within_dist), length(between_dist)))
  )
  ggplot(dist_df, aes(x = Type, y = Distance)) +
    geom_boxplot() +
    labs(title = "Within vs Between Subject Distances")
  # Formal test (approximate, for illustration)

}
```

- No repeated measures, no significant autocorrelation, no significant batch effect: Independence holds.

### Assumption 2: Homogeneity of Multivariate Dispersions
This is the assumption you already observed as unequal spread on PCoA plots. It is the second most critical, as unequal dispersions bias the PERMANOVA F-statistic (the ratio of between-group to within-group sum of squares).
#### What it means:
The average distance of samples to their group centroid (in Bray-Curtis space) is equal across all groups. For your design, test dispersions for:
1. The 6 Treatment×Time interaction groups (most important, as this is your key hypothesis)
2. Main effects of Treatment and Time (if interpreting main effects)
#### How to test:
Use `betadisper` (calculates group dispersions) + `permutest` (permutation-based test for dispersion differences, no normality assumption):
```r
# Test dispersions for Treatment×Time interaction groups
dispersion_interaction <- betadisper(d = bc_dist, group = interaction(meta$Treatment, meta$Time))
dispersion_test_interaction <- permutest(dispersion_interaction, permutations = 999)
print(dispersion_test_interaction)
# Optional: Test main effects separately
dispersion_trt <- betadisper(bc_dist, group = meta$Treatment)
dispersion_test_trt <- permutest(dispersion_trt, permutations = 999)

#### Interpretation:
- **Non-significant dispersion test (p > 0.05) + roughly equal CIs**: Homogeneity of dispersions holds. Proceed with standard PERMANOVA.
- **Significant dispersion test (p < 0.05) + visibly unequal CIs**: Dispersions are unequal. The severity of bias depends on two factors:
  1. Magnitude of dispersion difference: Differences <1.5x between groups are usually tolerable, especially with equal sample sizes (your 10 samples per cell reduces bias).
  2. Correlation with centroid differences: If the group with the smallest/largest dispersion is *not* the group with the largest centroid difference from other groups, bias is minimal. For example, if your tight-clustering group is the control, and treatment groups are more spread out but have centroids far from control, the test of Treatment effects is still valid. Bias is worst if groups with large centroid differences also have very different dispersions.
#### What to do if violated:
1. **Minor/moderate differences (no correlation with centroid differences)**: Proceed with PERMANOVA, but explicitly report the dispersion test results and note the limitation in your methods.
2. **Large differences or correlation with centroid differences**:
   - Transform your raw abundance data before calculating Bray-Curtis distances to reduce dispersion differences (common for microbiome data):
     ```r

     abund_sqrt <- sqrt(abund_table)
     # Recalculate Bray-Curtis distance
     bc_dist_sqrt <- vegdist(abund_sqrt, method = "bray")

     permutest(dispersion_interaction_sqrt, permutations = 999)
     ```
     Other transformations: log(abund + 1), or centered log-ratio (CLR) transformation followed by Euclidean distance (Aitchison distance, better for compositional data).
   - Remove outliers driving extreme dispersion (only if justified, e.g., sequencing failure).
   - Use a dispersion-robust alternative:
     - **MRPP (Multi-Response Permutation Procedure)**: Less sensitive to unequal dispersions, tests for group location differences. Use `mrpp()` in `vegan`:
       ```r
       mrpp_results <- mrpp(bc_dist ~ Treatment * Time, data = meta, permutations = 9999)

       ```
     - **Welch-type PERMANOVA**: Available in the `RVA` package, adjusts for unequal dispersions.

## 4. Key Limitations of Assumption Tests
1. The `betadisper` dispersion test has low power for small sample sizes (<5 per group), so a non-significant result does not guarantee homoscedasticity—always pair the test with visual PCoA diagnostics.
2. Independence is primarily a design-based assumption: formal tests for autocorrelation have low power unless sample size is very large.
3. Minor dispersion differences (especially with equal sample sizes) rarely bias PERMANOVA results meaningfully, so avoid overcorrecting for small, irrelevant differences.
````
