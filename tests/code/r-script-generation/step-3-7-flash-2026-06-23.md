2. Each subject has exactly one treatment assignment (no cross-over between probiotic/placebo) and samples at all 3 timepoints (baseline, week4, week8).

4. Subject is the only repeated measures blocking factor (no additional nested random effects are present in the study design).
5. Input data has been preprocessed prior to analysis (e.g., contaminant filtering, taxonomic aggregation to desired level); this script only includes basic QC steps.
6. Library size variation across samples is not extreme enough to bias Bray-Curtis distances; if library sizes vary by >2x, rarefaction or variance-stabilizing transformation is strongly recommended (included as optional steps in the script).
