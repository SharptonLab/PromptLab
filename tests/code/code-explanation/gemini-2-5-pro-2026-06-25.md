### 3. Key Concepts

-   **Design Formula (`~`)**: A standard R syntax for defining statistical models. `y ~ x1 + x2` means you are modeling the response `y` based on the explanatory variables `x1` and `x2`.
-   **Multiple Testing Correction (`padj`)**: When you perform thousands of tests (one for each microbial feature), you are bound to get some false positives by chance. Methods like Benjamini-Hochberg (the default in DESeq2) are used to calculate an "adjusted p-value" (`padj`) which controls the False Discovery Rate (FDR). A `padj` of 0.05 means that you expect no more than 5% of the features you call significant to be false positives.
-   **Log2 Fold Change (LFC)**: A standard way to represent fold changes in biology. It's used because it's symmetric (a 2-fold increase is +1, a 2-fold decrease is -1) and compresses large values, making it easier to visualize. An LFC of 1 means a 2-fold change, an LFC of 2 means a 4-fold change, etc.
