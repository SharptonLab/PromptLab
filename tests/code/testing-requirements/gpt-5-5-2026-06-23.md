## Correct CLR definition

For a sample count vector:

\[
x = [x_1, x_2, ..., x_D]
\]

The centered log-ratio transform is:

\[
CLR(x_i) = \log(x_i) - \frac{1}{D}\sum_{j=1}^{D}\log(x_j)
\]

Equivalently:

\[
CLR(x_i) = \log\left(\frac{x_i}{g(x)}\right)
\]

where \(g(x)\) is the geometric mean of the sample.

Important: CLR is usually applied **within each sample**, not within each feature/taxon.

For a microbiome table where rows are samples and columns are taxa, CLR should be applied row-wise.

## Important issue: zeros

CLR cannot be computed directly on zero counts because:

```python
np.log(0) == -inf
```

You must handle zeros, for example by adding a pseudocount:

```python
counts + 1
```

or using a more specialized zero-replacement method.

## Reference implementation

```python
import numpy as np
import pandas as pd

def clr_transform(counts, pseudocount=1):
    """
    Apply CLR transformation to a samples x features count table.

    Parameters
    ----------
    counts : pandas.DataFrame or numpy.ndarray
        Rows are samples, columns are features/taxa.
    pseudocount : float
        Value added to counts before log transformation.

    Returns
    -------
    pandas.DataFrame or numpy.ndarray
        CLR-transformed data with same shape as input.
    """
    x = counts + pseudocount
    log_x = np.log(x)
    clr = log_x - log_x.mean(axis=1, keepdims=True)

    if isinstance(counts, pd.DataFrame):
        return pd.DataFrame(clr, index=counts.index, columns=counts.columns)

    return clr
```

log([10, 20, 40]) - mean(log([10, 20, 40]))

def test_clr_equal_values_are_zero():
